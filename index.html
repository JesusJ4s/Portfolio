Mis Proyectos

Proyecto Dirección General de Salud Ambiental

El sistema de gestión y optimización desarrollado para la DGSA, es un sistema que ayuda en la agilización de los procesos administrativos del departamento de Salud Ambiental. El sistema permite llevar un control del parque tecnológico de la institución, así como apoyar en las operaciones de gestión de equipos y procesos de mantenimiento, permitiendo conocer el estado de cada proceso en tiempo real. Dicha aplicación se desarrolló utilizando PHP, MySQL, CSS y JavaScript.

Sistema de Gestión procesos administrativos de Tienda Musical Nippon

El sistema de gestión de procesos administrativos de la tienda musical, se desarrolló utilizando PHP, MySQL, CSS y JavaScript, además de utilizar Amazón Web Services para implantar en sistema en Internet. El sistema permite la gestión de inventarios, pedidos y pagos, así como la gestión de clientes y productos. En este sentido, la aplicación permite la gestión de tareas y el seguimiento de los pedidos en tiempo real. La misma se encuentra en fases de pruebas y optimización debido a que en paralelo se realiza su sitio web para ayudar a llegar a más clientes a la tienda.

Nippon Instruments E-commerce

La tienda de instrumentos musicales, se desarrolló utilizando PHP, MySQL, CSS y JavaScript, además de utilizar Amazón Web Services para implantar en sistema en Internet. El objetivo del E-commerce es permitir a los clientes comprar instrumentos musicales en línea, así como gestionar sus pedidos y pagos. La aplicación permite la gestión de productos, categorías, favoritos y usuarios. Actualmente, el E-commerce se encuentra en fase de pruebas y se espera que esté disponible para los clientes en los próximos meses.
